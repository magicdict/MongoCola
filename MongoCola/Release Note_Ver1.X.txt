========================================================================
MongoCola Release Note
========================================================================
MongoCola is developed by Microsoft .NET (C#)
You need install .NET framework 4.6 to run it.
The C# Mongo driver is mongo offical C# driver 2.0.1（Legacy）
Test Based on MongoDB 3.1.5
MongoCola is free and open source.
